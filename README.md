# 📱 Contact Timeline App

A modern contact tracking web application.  
It works like a hybrid between a phone contact app and a social activity tracker, helping you monitor how you interact with your contacts over time.

---

## 🚀 Features

- 👥 Manage contacts
- 📞 Track communication history
- 📊 View interaction timeline (call, text, video)
- 📅 See when you last contacted someone
- 📈 Stats dashboard using charts (Recharts)
- 🎨 Clean, responsive UI with Tailwind + DaisyUI

---

## 🧠 Concept

This app focuses on tracking your communication patterns:
- You interact with contacts using different methods:
  - Call 📞
  - Text 💬
  - Video 📹
- Every interaction is stored in a timeline
- You can visually analyze your communication habits

---

## 🛠️ Tech Stack

- **Next.js** – React framework for frontend
- **Tailwind CSS** – Utility-first styling
- **DaisyUI** – UI components built on Tailwind
- **Recharts** – Data visualization (pie charts, stats)

---

## 📊 Dashboard

The stats page displays:
- Number of calls
- Number of texts
- Number of video interactions

This helps you understand how you communicate with your contacts.
